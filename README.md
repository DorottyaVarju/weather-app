# Weather App

This is a very simple weather application. The user can enter the name of a city. Until she/he enters it, only the inputfield is displayed on the screen.
![initial](https://github.com/DorottyaVarju/weather-app/assets/125662526/6979ac55-8967-4e8e-998a-ae41e6856127)

Afterwards the data corresponding to the city will be displayed.
